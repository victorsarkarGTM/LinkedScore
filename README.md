# LinkedScore

> AI-powered LinkedIn profile scoring for professionals and recruiters.

---

## The problem

LinkedIn has 1 billion users. Most profiles are noise.

Recruiters spend an average of 7 seconds scanning a profile before moving on.
Professionals spend hours updating their profiles with no idea whether the changes
actually move the needle — or which parts of the profile are costing them
opportunities.

There is no objective benchmark. No signal. Just gut feel on both sides.

---

## What LinkedScore does

LinkedScore analyses a LinkedIn profile and returns a structured score across
the dimensions that actually determine recruiter and hiring manager response:

| Dimension | What's measured |
|---|---|
| Headline strength | Specificity, keyword relevance, value proposition clarity |
| Summary / About | Hook quality, quantified achievements, call to action |
| Experience section | Metrics density, active vs passive language, recency |
| Skills & endorsements | Relevance to target role, endorsement credibility |
| Engagement signals | Content activity, connection depth, recommendations |
| Profile completeness | Missing sections, photo, banner, contact info |

Output: a score out of 100, a percentile ranking against role-matched profiles,
and a prioritised list of improvements ranked by impact.
